# Quick Start

Welcome to Easy Algo! This guide will help you get started with our platform quickly.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or higher installed
- A modern web browser
- Basic understanding of algorithms and data structures

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/easyalgo.git
cd easyalgo
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

## Features

- **Interactive Learning**: Learn algorithms through interactive visualizations
- **Practice Problems**: Solve algorithm problems with real-time feedback
- **Progress Tracking**: Track your learning progress
- **Community Support**: Get help from our community
